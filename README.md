# IT Help Desk — Mock Ticket Portfolio

## Overview
10 IT support tickets created and resolved in Freshdesk simulating 
real Level 1 help desk scenarios. Each ticket includes a professional 
user-facing reply and a detailed internal technical note.

## Platform
Freshdesk (free tier) — industry standard help desk ticketing system

## Tickets Completed

| # | Issue | Type | Priority | Status |
|---|-------|------|----------|--------|
| 1 | User locked out after failed login attempts | Account Access | High |  Resolved |
| 2 | Printer showing offline — level 3 | Hardware | Medium |  Resolved |
| 3 | No internet or network drive access | Network | High |  Resolved |
| 4 | PC extremely slow — 15min boot time | Software | Medium |  Resolved |
| 5 | Adobe Acrobat install — 3 workstations | Software | Low |  Resolved |
| 6 | Laptop won't turn on before presentation | Hardware | Urgent |  Resolved |
| 7 | Outlook cannot send emails | Software | High |  Resolved |
| 8 | Monitor flickering after desk move | Hardware | Medium |  Resolved |
| 9 | New employee onboarding setup | New Setup | Medium |  Resolved |
| 10 | Malware alert — suspicious popup | Security | Urgent |  Resolved |
